This is
Arpon OS

This edition is in alpha, it works but only on shell and without network connection and tools to install program easily (only dpkg, but it need to download the package with ALL dependencies before install the program. Trust me, this takes too much time, so this edition is to help us on development...

this development branch was created to build a system all made by the NFT, to give us some help and ideas, go to our discord !

Arpon OS contains Busybox to be able to enjoy lots of programs with the most performance possible. Later, arpon OS will have its own interface, and will want to be closely linked to the command line. Indeed, I would like to use the i3 desktop for maximum performance.

The development will be restarted soon to integrate IA witch will optimize the system in real time.

HISTORY

Development started in 2018 by Neckfire, the first usable version released in January 2021.

The philosophy of Arpon OS is to “be lightweight, fast and to get some posibilities like biggest linux distros.”

“Arpon OS needs to be extremely customizable without using too much ressources.”

to start, you must have:

a USB pendrive

an iso img burning software (like Etcher)

have access to your boot menu

or else use an emulator, like QEMU

install

use In the first case :

connect your usb pendrive to your computer

download the iso »

open your burning software (etcher) choose the usb key, the iso and flash

reboot your computer and open the boot menu, choose the usb pendrive and start!

In the second case :

download the iso and your emulator

In this example, I’ll use qemu:

open CMD / terminal and type the command:

qemu-system-x86_64 -m 512 [the path to the iso]

start!
